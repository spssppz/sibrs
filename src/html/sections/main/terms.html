• Оплата
    Оплату можно произвести любым удобным для Вас способом: платежные системы сайта, переводом на карту, наличными в офисе, безналичной оплатой на расчетный счет (Без НДС).
  • Бесплатная доставка
    Доставка серверов и СХД во все регионы России осуществляется бесплатно, доставка комплектующих оплачивается при получении по тарифам транспортной компании.
  • Для каждого сервера
    С каждым купленным сервером в комплект входят: винты для жестких дисков, кабель для каждого блока питания и сетевой кабель RJ45 для модуля управления и одного порта сетевой карты.
