Сервер Dell PowerEdge R7525

  • Процессор 2 x AMD EPYC 7F52 3.5 - 3.9 GHz 16 ядер 32 потоков кэш 256Мб
  • Оперативная память 256Gb (32Gb x 8шт) DDR4 ECC REG 3200Mhz
  • Жесткие диски 2 x Intel 1.92T SATA SSD + 2 x 8T SATA HDD + 2 x 2T NVMe SSD m.2
  • Корзина На 12LFF
  • Блок питания 2 x 1400W с горячей заменой
  • Форм-фактор 2U
  • Поколение процессоров AMD EPYC 7002/7003
  • Тип памяти DDR4

Сервер Dell PowerEdge R7525 — это высокопроизводительное решение для корпоративных ИТ-инфраструктур, которое обеспечивает надежность и масштабируемость. Оснащенный процессором AMD EPYC 7F52 с базовой частотой 3.5 ГГц и возможностью увеличения до 3.9 ГГц в режиме Turbo Boost, а также 16 ядрами, этот сервер предоставляет мощные вычислительные возможности для самых требовательных задач.
256 ГБ оперативной памяти позволяют серверу эффективно обрабатывать множество параллельных процессов и справляться с ресурсоемкими приложениями. Два SSD по 1.92 ТБ и два NVMe m.2 накопителя по 2 ТБ каждый обеспечивают высокую скорость загрузки и обработки данных, в то время как два SATA HDD по 8 ТБ предоставляют значительное пространство для долговременного хранения данных. Контроллер H745 обеспечивает надежную работу с данными, а два блока питания по 1400 Вт каждый гарантируют стабильную и бесперебойную работу системы.
